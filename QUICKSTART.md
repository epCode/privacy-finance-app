# Quick Start Guide

Get up and running with Privacy Finance App in minutes.

## Installation

### macOS and Linux

1. Download the latest release from GitHub Releases
2. For macOS: Double-click the `.dmg` file and drag Privacy Finance to Applications
3. For Linux: Download the `.AppImage` or `.deb` file
   - AppImage: Make executable and run: `chmod +x Privacy-Finance-*.AppImage && ./Privacy-Finance-*.AppImage`
   - Deb: Install with: `sudo dpkg -i Privacy-Finance-*.deb`

### Windows

1. Download the `.exe` installer from GitHub Releases
2. Run the installer and follow the setup wizard
3. Privacy Finance will launch automatically after installation

### Android

1. Download the `.apk` file from GitHub Releases
2. Transfer to your Android device
3. Open the file manager and tap the APK to install
4. Grant necessary permissions when prompted

## First Steps

### 1. Create Your First Account

1. Click on "Accounts" in the sidebar
2. Click "New Account"
3. Fill in the details:
   - **Account Name**: e.g., "My Checking Account"
   - **Account Type**: Select from Checking, Savings, Credit Card, Investment, or Cash
   - **Initial Balance**: Enter your current balance
   - **Currency**: USD (or your preferred currency)
4. Click "Create Account"

### 2. Add Your First Transaction

1. Click on "Transactions" in the sidebar
2. Click "New Transaction"
3. Fill in the details:
   - **Account**: Select the account you just created
   - **Type**: Choose Income or Expense
   - **Amount**: Enter the transaction amount
   - **Category**: e.g., "Groceries" or "Salary"
   - **Date**: Select the transaction date
   - **Note**: Optional description
4. Click "Add Transaction"

Your account balance will update automatically!

### 3. Set Up a Recurring Payment

1. Click on "Recurring Payments" in the sidebar
2. Click "New Payment"
3. Fill in the details:
   - **Account**: Select an account
   - **Payment Name**: e.g., "Monthly Rent"
   - **Amount**: Enter the payment amount
   - **Frequency**: Choose Daily, Weekly, Monthly, etc.
   - **Start Date & Time**: When the payment should start
   - **Constant Amount**: Toggle if the amount is always the same
   - **Default Amount**: The amount to process
4. For non-constant payments (e.g., utilities), add a custom notification message
5. Click "Create Payment"

### 4. Create a Budget

1. Click on "Budgets" in the sidebar
2. Click "New Budget"
3. Fill in the details:
   - **Category**: e.g., "Groceries"
   - **Spending Limit**: Your monthly/yearly limit
   - **Period**: Monthly or Yearly
4. Click "Create Budget"

Now you'll see your spending tracked against your budget!

### 5. Create a Transaction Template

1. Click on "Templates" in the sidebar
2. Click "New Template"
3. Fill in the details:
   - **Template Name**: e.g., "Weekly Groceries"
   - **Category**: e.g., "Groceries"
   - **Default Amount**: Your typical spending
   - **Note**: Optional description
4. Click "Create Template"

Later, when adding transactions, click "Use Template" to quickly fill in the form!

## Common Tasks

### Check Your Dashboard

Click "Dashboard" to see:
- Total balance across all accounts
- Monthly income and expenses
- Recent transactions
- Upcoming recurring payments
- Any pending notifications

### Edit a Transaction

1. Go to "Transactions"
2. Find the transaction you want to edit
3. Click the edit icon (pencil)
4. Make your changes
5. Click "Update Transaction"

### Delete a Transaction

1. Go to "Transactions"
2. Find the transaction you want to delete
3. Click the delete icon (trash)
4. Confirm the deletion

### Process a Recurring Payment Manually

1. Go to "Recurring Payments"
2. Find the payment you want to process
3. Click the process button (if available)
4. For non-constant payments, enter the actual amount
5. Confirm

### View Your Spending by Budget

1. Go to "Budgets"
2. See the progress bar for each category
3. Red indicates overspending
4. Click on a budget to see details

## Tips and Tricks

**Use Templates for Speed**: Create templates for your most common transactions to save time.

**Set Realistic Budgets**: Start with your actual spending patterns, then adjust as needed.

**Review Regularly**: Check your Dashboard weekly to stay on top of your finances.

**Organize with Categories**: Use consistent category names to make budgeting and reporting easier.

**Backup Your Data**: Your data is stored locally on your device. Back up your computer regularly to protect your financial records.

**Non-Constant Payments**: Use these for bills that vary (utilities, phone) so you're reminded to verify the amount each month.

**Use Notes**: Add notes to transactions for context, especially for large or unusual expenses.

## Troubleshooting

### App Won't Start

- Restart your computer
- Try reinstalling the app
- Check that you have sufficient disk space

### Data Disappeared

- The app stores data locally on your device
- Check if you've cleared your browser cache (for web version)
- Restore from a backup if available

### Recurring Payment Not Processing

- Check the start date and time
- Verify the frequency is set correctly
- Ensure the account still exists
- Check notifications for any pending actions

### Performance Issues

- Close other applications to free up memory
- Restart the app
- Clear old transactions if you have thousands of records
